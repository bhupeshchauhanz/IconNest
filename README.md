# IconNest — Premium Icon Library 🪺

> A curated, browser-based icon gallery featuring **2,500+ premium icons** across four collections. Browse, search, preview, and download individual icons or entire style packs — all in one place.

---

## ✨ Live Preview

For the best experience (required for single-file SVG downloads to work properly), open `index.html` via **VS Code Live Server** or any local web server.

---

## 📦 Collections

- **Iconly:** 113 UI icons in varied styles (Curved, Regular, Sharp) and weights (Light, Bold, Broken, etc.)
- **Glassy Icons:** 50 icons with a frosted-glass aesthetic
- **3D Shape Icons:** 61 shapes in multiple rendering styles and angles
- **Weather Icons:** 25 conditions available in animated, static, and 3D formats

---

## 🚀 Features

- **Universal Search:** Search across all collections simultaneously.
- **Style Switcher:** Easily toggle between different weights, angles, and rendering styles.
- **Instant Downloads:** Download individual icons or bulk pre-built ZIP style packs.
- **Premium UI:** Animated 3D-tilt hover effects, glassmorphic interactions, and smooth transitions.

---

## 🛠️ Tech Stack

- HTML5
- Vanilla CSS
- Vanilla JavaScript
- [JSZip](https://stuk.github.io/jszip/)

*No heavy frameworks, no build steps required.*
